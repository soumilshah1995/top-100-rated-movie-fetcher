# MovieFetcher

This Library allows Developers to Get all top 100 Movies with Rating in JSON and CSV File 

## Installation

```bash
pip install moviefetcher
```
## Usage

```python
import sys
from moviefetcher.moviefetcher import MovieRatings

    obj = MovieRatings(year=2019)
    obj.saveAsCsv()
    obj.saveAsJson()


```
